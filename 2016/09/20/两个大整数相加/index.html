两个大整数相加

发表于 2016-09-20

题目

现有两个非常大的整数, 不论使用 int / long / long long 都无法存储, 该如何相加?

思路

假设有两个整数 a 和 b , a 的位数大于 b, 那么 a + b 的位数最多为 a + 1,
也就是说两个整数相加的结果的位数最多比较大的那个整数的位数大1.

因此, 我们可以使用拆分法, 把两个整数拆分成多段, 分段相加.

比如

a = 123456789 b = 5555555

我们把整数拆分成 3 位数一段, 那么 a = 123/456/789, b = 5/555/555.

分段相加, c = 123+5 / 456+555 / 789+555 = 128/1011/1344

我们发现 456+555=1011 和 789+555=1344 发生了进位, 所以我们把进位的 1 和高段相加, 128+1=129, 011+1=012 所以最终结果就为 129012344
